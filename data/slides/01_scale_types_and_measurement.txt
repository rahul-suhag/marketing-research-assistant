MARKETING RESEARCH: SCALE TYPES AND MEASUREMENT LEVELS
=======================================================

1. NOMINAL SCALE
-----------------
Definition: A nominal scale classifies data into distinct categories with no inherent order or ranking. Categories are mutually exclusive and collectively exhaustive.

Examples in Marketing Research:
- Gender (Male, Female, Non-binary, Prefer not to say)
- Brand preference (Nike, Adidas, Puma, Under Armour)
- Geographic region (North, South, East, West)
- Purchase channel (Online, In-store, Mobile app)
- Yes/No questions (Did you purchase in the last 30 days? Yes/No)
- Occupation categories
- Marital status

Properties:
- Categories have no mathematical relationship
- Cannot calculate mean or median meaningfully
- Can only count frequencies

How to Clean Nominal Data in Qualtrics Exports:
- Check for inconsistent spellings (e.g., "male" vs "Male" vs "MALE")
- Standardize category labels to a consistent format
- Identify and handle "Other" responses
- Remove or recode blank/missing responses
- Binary coding: Convert Yes/No to 1/0 using Excel IF function or pandas .map()

Binary Coding for Nominal Variables:
- For Yes/No: =IF(A2="Yes",1,0)
- For Gender dummy coding: Create separate columns for each category
  - Male: =IF(A2="Male",1,0)
  - Female: =IF(A2="Female",1,0)
  - One category must be omitted as the reference category in regression

Appropriate Statistical Analyses for Nominal Data:
- Frequency distribution (counts and percentages)
- Mode (most frequent category)
- Chi-square test of independence (to test association between two nominal variables)
- Cross-tabulation (contingency tables)
- Cramer's V (measure of association strength)
- Binomial test (for binary nominal variables)
- Lambda (measure of proportional reduction in error)

NOT Appropriate for Nominal Data:
- Mean, median
- Standard deviation
- Correlation
- t-test
- ANOVA
- Regression (unless dummy coded)


2. ORDINAL SCALE
-----------------
Definition: An ordinal scale classifies data into categories that have a meaningful order or ranking, but the intervals between categories are not necessarily equal.

Examples in Marketing Research:
- Likert scales (Strongly Disagree, Disagree, Neutral, Agree, Strongly Agree)
- Satisfaction ratings (Very Dissatisfied to Very Satisfied)
- Education level (High School, Bachelor's, Master's, PhD)
- Income brackets (Under $25K, $25K-$50K, $50K-$75K, $75K+)
- Purchase frequency (Never, Rarely, Sometimes, Often, Always)
- Net Promoter Score categories (Detractor, Passive, Promoter)
- Ranking questions (Rank your top 3 preferred brands)

Properties:
- Categories have a meaningful order
- Distances between categories are not guaranteed to be equal
- "Agree" is higher than "Disagree" but the difference is not quantifiable

How to Clean Ordinal Data in Qualtrics Exports:
- Verify that response codes match the intended order (1=Strongly Disagree through 5=Strongly Agree)
- Check for reverse-coded items and recode them (e.g., if 1=Strongly Agree, recode to 5)
- Handle missing responses: decide between listwise deletion, pairwise deletion, or imputation
- Recode text responses to numerical values if not already coded
- Example Excel formula for recoding: =IF(A2="Strongly Disagree",1,IF(A2="Disagree",2,IF(A2="Neutral",3,IF(A2="Agree",4,IF(A2="Strongly Agree",5,"")))))

Appropriate Statistical Analyses for Ordinal Data:
- Frequency distribution
- Median and mode
- Percentiles and quartiles
- Mann-Whitney U test (non-parametric alternative to t-test)
- Wilcoxon signed-rank test (paired comparison)
- Kruskal-Wallis test (non-parametric alternative to ANOVA)
- Spearman rank correlation
- Kendall's tau
- Cross-tabulation with chi-square

Note on Likert Scales:
Many researchers treat 5-point and 7-point Likert scales as interval data when conducting analysis such as computing means and running t-tests or ANOVA. This is common practice in marketing research but should be noted as an assumption. When Likert items are combined into a composite scale (summated rating scale), the resulting scale is more commonly treated as interval.

NOT Appropriate for Ordinal Data (strictly):
- Arithmetic mean (controversial; see note above)
- Standard deviation (controversial; see note above)
- Pearson correlation (use Spearman instead)
- Parametric tests without careful justification


3. INTERVAL SCALE
------------------
Definition: An interval scale has ordered categories with equal intervals between values, but no true zero point. The zero is arbitrary.

Examples in Marketing Research:
- Temperature scales (Fahrenheit, Celsius)
- Likert scale composites (when treated as interval after summation)
- Semantic differential scales (e.g., rate on a scale from 1 to 7 where 1=Bad and 7=Good)
- Year of birth / calendar dates
- Standardized test scores (SAT, GRE)
- Attitude indices computed from multiple Likert items

Properties:
- Equal intervals between values
- Addition and subtraction are meaningful
- Ratios are NOT meaningful (80 degrees F is not "twice as hot" as 40 degrees F)
- No true zero point

How to Clean Interval Data:
- Check for out-of-range values (e.g., a 1-7 scale with a value of 8 or 0)
- Identify outliers using the IQR method or z-scores
- Standardize or normalize if comparing across different scales
- Check for normality using histograms or Shapiro-Wilk test
- Handle missing data: mean imputation, regression imputation, or deletion

Appropriate Statistical Analyses for Interval Data:
- Mean and standard deviation
- Pearson correlation coefficient
- Independent samples t-test (compare two group means)
- Paired samples t-test (compare means before/after)
- One-way ANOVA (compare means across 3+ groups)
- Two-way ANOVA (examine main effects and interactions)
- Linear regression
- Factor analysis
- Reliability analysis (Cronbach's alpha)
- Cluster analysis

Excel Tools for Interval Data:
- Data Analysis ToolPak: Descriptive Statistics, t-Test, ANOVA, Correlation, Regression
- Functions: AVERAGE(), STDEV(), CORREL(), T.TEST(), F.TEST()
- Pivot Tables for group comparisons


4. RATIO SCALE
---------------
Definition: A ratio scale has all properties of an interval scale plus a true zero point, making ratios meaningful.

Examples in Marketing Research:
- Age (in years)
- Income (in dollars)
- Number of purchases
- Amount spent
- Time spent on website (in seconds/minutes)
- Distance traveled to store
- Number of brand mentions on social media
- Survey duration (in seconds)
- Household size
- Weight, height

Properties:
- True zero point (zero means absence of the quantity)
- Ratios are meaningful (someone who spent $100 spent twice as much as someone who spent $50)
- All mathematical operations are valid

How to Clean Ratio Data:
- Check for negative values where none should exist (e.g., negative age)
- Check for impossible values (e.g., age = 200)
- Identify and handle outliers
- Check for zero values that might indicate missing data
- Log-transform highly skewed variables if needed for parametric tests
- Check for normality

Appropriate Statistical Analyses for Ratio Data:
- All analyses valid for interval data, PLUS:
- Geometric mean
- Coefficient of variation
- Ratio comparisons
- All parametric and non-parametric tests


5. COMPARATIVE VS. NON-COMPARATIVE SCALES
-------------------------------------------

Scales used in marketing research fall into two broad families: comparative scales and non-comparative scales.

5A. COMPARATIVE SCALES
Comparative scales ask respondents to evaluate objects relative to each other. They directly compare two or more stimuli.

Types of Comparative Scales:

a) Paired Comparison Scale
- Respondents choose between two objects at a time (e.g., "Do you prefer Brand A or Brand B?").
- For n brands, the respondent makes n(n-1)/2 comparisons.
- Useful when the number of objects is small.
- Limitation: becomes cumbersome with many objects.

b) Rank-Order Scale
- Respondents rank a set of objects from most to least preferred (e.g., "Rank these 5 brands from 1=Most Preferred to 5=Least Preferred").
- Produces ordinal data.
- Forces respondents to discriminate among objects.
- Limitation: only produces ordinal data; cannot tell how much more one item is preferred over another.

c) Constant-Sum Scale
- Respondents allocate a fixed number of points (typically 100) among several attributes or objects to indicate relative importance.
- Example: "Allocate 100 points among the following product features based on importance: Quality ___, Price ___, Design ___, Durability ___."
- Produces ratio-level data for the relative importance of attributes.
- Advantage: forces respondents to make trade-offs.
- Limitation: may be difficult for respondents if too many attributes; points must sum to the total.

Comparative scales are useful when you want to understand relative preferences but they do not measure absolute attitudes.

5B. NON-COMPARATIVE SCALES
Non-comparative scales (also called monadic or metric scales) evaluate objects one at a time without direct comparison to other objects.

Types of Non-Comparative Scales:

a) Likert Scale
- A summated rating scale where respondents indicate their level of agreement or disagreement with a series of statements.
- Typically uses 5 or 7 points: Strongly Disagree, Disagree, Neither Agree nor Disagree (Neutral), Agree, Strongly Agree.
- Each statement is called a "Likert item"; combining multiple items creates a Likert scale (composite/summated scale).
- Scale points are the numbers assigned to each response option (e.g., 1 through 5 or 1 through 7).
- Produces ordinal data at the item level; treated as interval when items are combined into a composite score.
- Very widely used in marketing research for measuring attitudes, opinions, and perceptions.
- Example: "I am satisfied with Brand X's customer service." (Strongly Disagree 1 2 3 4 5 Strongly Agree)

b) Semantic Differential Scale
- A scale anchored by bipolar adjective pairs at opposite ends.
- Respondents mark their position on a continuum between two opposites.
- Typically uses 7 points.
- Only the endpoints of the scale are labeled (the bipolar adjectives).
- Commonly used to measure attitudes, brand image, and perceptions.
- Example: Unfriendly 1 2 3 4 5 6 7 Friendly
- The mean for each attribute can be calculated and plotted as a profile.
- Difference from Likert: Likert measures agreement with statements; semantic differential measures position between two opposite adjectives.

c) Graphic Rating Scale (Continuous Rating Scale)
- Respondents indicate their rating by placing a mark on a continuous line or slider between two endpoints.
- Example: a slider from "Very Unsatisfied" to "Very Satisfied".
- Can produce interval-level data.
- Advantage: high granularity; respondents not forced into discrete categories.

d) Behavioral Intention Scale
- Measures respondents' likelihood of performing a future behavior.
- Example: "How likely are you to purchase this product in the next 6 months?" (Definitely will not buy 1 2 3 4 5 Definitely will buy)

Comparison: Likert vs. Constant-Sum Scale
- Likert Scale: Measures individual agreement/disagreement with statements; each item rated independently; produces ordinal/interval data; does not require trade-offs between attributes.
- Constant-Sum Scale: Requires allocation of a fixed total (e.g., 100 points) across attributes; forces trade-offs; produces ratio data for relative importance; compares attributes directly.
- Key difference: Likert measures absolute attitudes per statement; constant-sum measures relative importance by forcing allocation of limited points.

Comparison: Likert vs. Comparative Scale
- Likert Scale is a non-comparative scale: each object/statement is evaluated independently, not relative to others.
- Comparative scales (paired comparison, rank-order, constant-sum) require evaluation of objects relative to each other.
- Key difference: Likert gives absolute attitude measurement; comparative scales reveal relative preferences and force discrimination between objects.

Comparison: Likert vs. Semantic Differential
- Both are non-comparative scales used to measure attitudes.
- Likert uses agree/disagree response format with declarative statements.
- Semantic differential uses bipolar adjective pairs with only endpoints labeled.
- Likert is better for measuring agreement with specific propositions; semantic differential is better for measuring brand image or perceptions along multiple dimensions.


6. CONSTRUCTS AND CONSTRUCT DEVELOPMENT
-----------------------------------------

A construct is a hypothetical variable made up of a set of component responses or behaviors that are thought to be related. Constructs are abstract concepts that cannot be directly observed (e.g., brand loyalty, customer satisfaction, purchase intention, perceived quality).

Construct Development Process:
1. Begin with an accurate definition of the purpose of the study.
2. Define the construct clearly and precisely.
3. Identify indicator variables (observable items) that reflect the construct.
4. Develop scale items (questions) to measure each indicator.
5. Test for reliability (Cronbach's alpha) and validity.

Key points about constructs:
- Constructs are NOT directly measured; they are inferred from multiple indicator variables.
- Constructs are NOT directly observable; they are latent (hidden) variables.
- A single survey question rarely captures a full construct; multiple items are needed.
- Example: "Customer Satisfaction" might be measured by 5 Likert items covering different aspects of satisfaction.


7. SCALE DESIGN DECISIONS
---------------------------

a) Number of Scale Points
- Common options: 5-point, 7-point, 9-point, 10-point.
- More points = finer discrimination but may overwhelm respondents.
- Fewer points = easier to answer but less precision.

b) Balanced vs. Unbalanced Scale
- Balanced: Equal number of positive and negative categories (e.g., 2 negative + neutral + 2 positive = 5-point balanced).
- Unbalanced: Unequal positive/negative categories (e.g., 1 negative + 3 positive). Can bias results.
- Best practice: Use balanced scales for unbiased measurement.

c) Forced vs. Non-Forced Scale
- Forced: No neutral or "Don't Know" option; respondents must lean positive or negative. Even-numbered scales (e.g., 4-point or 6-point) with no midpoint are forced scales.
- Non-forced: Includes a neutral midpoint or "Don't Know / Not Applicable" option.
- Forced scales push respondents to take a position; non-forced scales allow genuinely neutral responses.
- Drawback of forced scales: respondents with no real opinion may guess, producing inaccurate data.
- Drawback of not offering "Don't Know": respondents unfamiliar with the topic may provide uninformed answers.

d) Single-Item vs. Multiple-Item Scale
- Single-item scale: Collects data about only one attribute of the object being investigated. One question measures one thing.
- Multiple-item scale: Uses several questions to measure different facets of the same construct. More reliable and valid.
- Example: Single-item = "Overall, how satisfied are you?" vs. Multiple-item = 5 questions covering different dimensions of satisfaction.

e) Scale Descriptors
- Labels placed at scale points to help respondents understand the meaning.
- Can be verbal (word labels at each point), numerical (just numbers), or a combination.
- Scale responses should be mutually exclusive (no overlapping categories).


8. SURVEY QUESTION DESIGN PITFALLS
-------------------------------------

a) Double-Barreled Questions
- Combine two or more attributes in one question but allow only a single answer.
- Example (bad): "How satisfied are you with our product quality and customer service?"
- Problem: Respondent may feel differently about quality vs. service but can only give one answer.

b) Leading Questions
- Suggest a socially desirable or expected answer.
- Example (bad): "Don't you agree that our service is excellent?"
- Problem: Biases responses toward agreement.

c) Double Negative Questions
- Use two negations, creating cognitive confusion.
- Example (bad): "I would not say that the product is not worth buying."
- Problem: Hard to interpret; respondents may misunderstand the intended meaning.

d) Loaded Questions
- Contain emotionally charged words that influence responses.
- Example (bad): "Do you support the wasteful government spending on this program?"

6. CHOOSING THE RIGHT ANALYSIS FOR YOUR MRP (Marketing Research Project)
-------------------------------------------------------------------------

Step 1: Identify your research questions and hypotheses.
Step 2: Identify the scale type of each variable involved.
Step 3: Determine if you are examining:
  - One variable (univariate): Use descriptive statistics appropriate to the scale
  - Two variables (bivariate): Use association/comparison tests appropriate to both scales
  - Multiple variables (multivariate): Use regression, factor analysis, or cluster analysis

Common MRP Analysis Scenarios:

Scenario A: "Is there a difference in satisfaction between males and females?"
- IV: Gender (Nominal)
- DV: Satisfaction (Ordinal/Interval)
- If treating satisfaction as interval: Independent samples t-test
- If treating satisfaction as ordinal: Mann-Whitney U test
- Excel: Data > Data Analysis > t-Test: Two-Sample Assuming Unequal Variances

Scenario B: "Is there a relationship between age and purchase amount?"
- Both variables are Ratio
- Use: Pearson correlation, scatter plot, simple linear regression
- Excel: Data > Data Analysis > Correlation, or Data > Data Analysis > Regression

Scenario C: "Is there an association between gender and brand preference?"
- Both variables are Nominal
- Use: Cross-tabulation with chi-square test
- Excel: Create a Pivot Table, then manually calculate chi-square or use add-in

Scenario D: "Do satisfaction scores differ across 4 brand groups?"
- IV: Brand group (Nominal with 4 categories)
- DV: Satisfaction (Interval)
- Use: One-way ANOVA, followed by post-hoc tests (Tukey HSD)
- Excel: Data > Data Analysis > Anova: Single Factor

Scenario E: "What factors drive purchase intention?"
- DV: Purchase intention (Interval)
- IVs: Multiple (various scales, dummy-code nominal)
- Use: Multiple linear regression
- Excel: Data > Data Analysis > Regression (include all IVs)
